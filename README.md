# SOS-button
it's a button which can send your live location in emergency . its a really useful project which made from two microcontroler 
and it use a button to send your live location by asking help to your saved num.

<img width="470" height="369" alt="Screenshot 2026-07-17 133233" src="https://github.com/user-attachments/assets/ad80e63d-02c7-4e23-8f15-04e7ab4153bb" />

this is really useful for those teenager studients who don't carry any smart phone or any other stuff so that they can make contact with parents . by just one click it can send your live location and your parents can alert that you need help.to make this project a xiaoesp32 which is basically the main brain of this project and for long range connectionsi use a A9G Core Board Module GPRS+GSM Development Board and for power the whole board i use a lipo battery with a tp4056 charging module to charge the battery.
